    It is an information retrieval system that enables users to search information on the Internet in hypertext and multimedia environments. It is a system that effectively searches various information on the Internet expressed in hypertext format and is the most widely used worldwide. [One] Hypertext is displayed in the form of reading pieces of information such as “documents” or web pages from a web server through a program called a web browser and displaying them on a computer monitor . The user can then follow the hyperlinks on each page to navigate to other documents or send a series of information to the server serving the page. The act of moving along a hyperlink is commonly referred to as web surfing ( cultural language : network tour [2] ) or web browsing . And a collection of web pages with related contents is called a web site . The English word worldwide is usually written as a single word without spaces or hyphens, but the World Wide Web and its abbreviation WWW are used as official English words. The World Wide Web can be summarized in three functions: The first unified method of locating web resources, eg URL . The protocol for accessing resource names on the second web, for example HTTP . A language that allows easy navigation between third resources, such as HTML .
